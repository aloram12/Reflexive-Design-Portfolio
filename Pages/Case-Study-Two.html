Why HIPAA Compliance Mattered

After speaking with our client, I kept thinking about what could go wrong if someone’s private medical information accidentally got sent to the wrong person. Imagine someone submitting a request for medical leave and that request with all their private details ends up being shared across 13 different Air Force units.

"This wasn’t just about file organization. It was about protecting people."

The Air Force needed a system that could encourage communication while making sure private information stayed safe. HIPAA added complexity to the design but it was essential. This part of the process gave me a deeper respect for secure systems planning.

Figure 1. Hierarchy of Air Force departments and where medical groups sit in the structure.

Organizing Tasks with Clarity

We used SharePoint Lists to organize tasks into familiar spreadsheet-style views, breaking them down by unit or department. Then, using Power Automate, we synchronized each unit’s list into a master list for admin-level visibility only.

This ensured that leadership could track progress, while keeping sensitive departmental tasks private.

Automating Privacy with Power Automate

We built secure workflows that automatically routed sensitive documents to specific recipients and restricted access based on department affiliation.

  • When someone adds a document, the “Unit” field is auto-filled
  • Access is granted only to that unit’s members
  • Others are restricted from viewing it

Figure 2. Automated flow logic for department-specific access control.

Challenges with Microsoft Teams Integration

At first, it seemed ideal to use Microsoft Teams channels because each one creates its own SharePoint site. But this quickly became fragmented. These SharePoint “islands” didn’t talk to each other well, and trying to connect them slowed down collaboration.

"What we pictured as a city with locked buildings turned into a set of isolated islands."

This led us to redesign the structure so teams could stay private while still being able to communicate and share data when needed like separate houses in the same neighborhood.

Figure 3. Teams application interface where a SharePoint is embedded for internal collaboration.

Testing and Real-World Constraints

Since we only had student-level Microsoft 365 accounts, we couldn’t use premium features like Azure Active Directory. So we recreated the access logic manually through Power Automate flows.

It was slow and often required troubleshooting, but it worked—and it helped us learn just how much you can accomplish with determination and resourcefulness.
